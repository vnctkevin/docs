  # Register an asset

post https://api.fireblocks.io/v1/assets

Register a new asset to a workspace and return the newly created asset's details. Currently supported chains are:

*   EVM based chains
*   Stellar
*   Algorand
*   TRON
*   NEAR
*   Solana
